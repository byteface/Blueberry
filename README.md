# Blueberry π

## setup

mac/linux:

```bash
python3 -m venv venv
. venv/bin/activate
python3 -m pip install -r requirements.txt
```

windows:

```bash
python3 -m venv venv
venv\Scripts\activate
python3 -m pip install -r requirements.txt
```

## running

```bash
python3 blueberry.py
```

## customising

edit the congi.ini and comment out the things you don't want to appear in the Dock
